# HP Laptop Manager v1.0.0 — Release Notes

## 🔴 Critical Fixes

### 1. Fixed `tomllib` Crash
The application crashed on startup on Python 3.10 and below if the `tomli` package was missing. Implemented a triple-fallback: `tomllib` → `tomli` → JSON-only mode.

### 2. Fan RPM Persistence & Safety
Previously, custom fan RPM targets weren't saved, potentially leaving fans at 0 RPM after a reboot in custom mode. Now, the fan mode safely falls back to `auto` during boot to prevent thermal issues.

### 3. `_find_hwmon_by_name` Crash Guard
The Fan page crashed with a `FileNotFoundError` if the `/sys/class/hwmon` directory was missing (e.g., in container environments). Added a directory existence check.

### 4. Secure TOML Configuration Writing
Manual TOML writing could be corrupted by special characters. Values are now sanitized, and a JSON fallback config is also written for better compatibility.

---

## 🟡 Medium Improvements

### 5. Thread-Safe `save_state()`
The daemon's `state` dictionary is now deep-copied before being written to disk, preventing race conditions during concurrent thread access.

### 6. MUX Page Reboot Feedback
If the `systemctl reboot` command fails, an error message is now displayed in the UI instead of failing silently.

### 7. Sensor Category Translation
The "Other" sensor category is now properly translated based on the language setting (formerly hardcoded as "Diğer").

### 8. Fan Curve Axis Translation
"Temperature (°C)" and "Fan Speed (%)" axis labels are now translated using the `T()` function.

### 9. Power Profile Tooltip Translation
Descriptions for Power Saver, Balanced, and Performance profiles are now displayed in the selected language.

### 10. Dashboard Temperature Unit Support
The dashboard now respects the user's preference for °C or °F (previously hardcoded to °C).

### 11. Version Comparison Suffix Support
Version tags like `0.9.0-rc2` are now correctly compared. Pre-release tags are treated as lower versions than the final release.

---

## 🟢 Cosmetic & Minor Fixes

### 12. Bare `except` Patterns Cleaned
Updated multiple instances of `except: pass` to `except Exception: pass` in `tools_page.py` to avoid swallowing system signals.

### 13. Duplicate Import Cleanup
Removed a redundant `import sys, os` line in `games_page.py`.

### 14. Battery Health Percentage Format
Fixed battery health display format from `%85` to `85%` for better international consistency.
